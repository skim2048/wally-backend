# 1. Introduction (개요)

## 1.1 Purpose (목표)

- 본 문서는 wally-backend의 소프트웨어 요구사항을 정의한다.
- 본 문서의 목적은 이 프로젝트가 충족해야 할 사항을 설계와 구현에 앞서 확정한다.
- 확정된 요구사항은 설계·구현의 기준이자 검증의 근거가 되며, 각 요구사항에 식별자를 부여하여 이후 산출물·시험과 추적 가능하게 매치한다.
- 기존 프로토타입 babycat을 참조하여 wally-backend를 처음부터 다시 설계하는 작업의 기준선 역할을 한다.
- 제품의 범위·용도는 1.2에서, 대상 독자는 1.6에서 규정한다.
